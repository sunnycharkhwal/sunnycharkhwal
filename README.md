### 🌍 Wanderlust — Production-Grade DevSecOps Platform

> *A travel app transformed into a fully automated, cloud-native, security-hardened delivery system on AWS EKS.*

<table>
<tr>
<td width="50%" valign="top">

**🔧 Infrastructure**
- `Terraform` — EKS cluster, VPC, IAM roles, S3, ECR, all version-controlled
- `Helm` — Kubernetes manifests templated for multi-env deploys
- `Ansible` — Node configuration management & bootstrapping
- `AWS Secrets Manager` — Zero plaintext secrets in codebase

</td>
<td width="50%" valign="top">

**🔄 Delivery**
- `Jenkins` — Multi-stage pipeline: build → test → scan → push → deploy
- `ArgoCD` — Continuous GitOps sync, self-healing cluster state
- `GitHub Actions` — PR checks, lint, test gates
- `Azure DevOps` — Cross-team pipeline integration

</td>
</tr>
<tr>
<td width="50%" valign="top">

**🔒 Security Gates at Every Stage**
- `SonarQube` — Static code analysis, quality gate enforcement
- `Trivy` — Container image vulnerability scanning (blocks CVEs)
- `OWASP` — Dependency-check against known exploits
- `IAM + RBAC` — Least-privilege access across all services

</td>
<td width="50%" valign="top">

**📊 Observability**
- `Prometheus` — Metrics scraping from all pods & nodes
- `Grafana` — Custom dashboards: CPU, memory, request rates, latency
- `CloudWatch` — AWS-native log aggregation + SNS alerting
- `Alertmanager` — PagerDuty-ready threshold-based alerts

</td>
</tr>
</table>

<summary>⚙️ <b>One-time setup — click to enable the snake animation</b></summary>

Create this file in your profile repo at `.github/workflows/snake.yml`:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 0 * * *"   # runs daily at midnight UTC
  workflow_dispatch:        # lets you trigger it manually too

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk/svg-only@v3
        with:
          github_user_name: sunnycharkhwal
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

After pushing this file, go to **Actions → Generate Snake Animation → Run workflow** once manually. The snake SVG will appear instantly.
